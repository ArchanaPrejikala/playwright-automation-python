# Playwright Automation Framework (Python)

## Overview
This project demonstrates browser automation using Playwright with Python.
It includes positive and negative test scenarios for form validation.

## Tools Used
- Python
- Playwright
- VS Code

## Test Scenarios
- Positive form submission validation
- Negative validation for invalid email input

## How to Run
1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Install browsers:
   playwright install
4. Run tests:
   python tests/test_positive_form.py
   python tests/test_negative_form.py
